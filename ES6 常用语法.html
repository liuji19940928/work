ES6 常用语法

ECMAScript 6（简称ES6）是JavaScript语言的新一代标准。因为当前版本的ES6是在2015年发布的，所以又称ECMAScript 2015。
注意: ES6部分语法目前还不被浏览器支持,需要使用Babel进行转码。

let, const

let和const用途与var类似，都是用来声明变量的。
let声明的变量可以改变，值和类型都可以改变，没有限制，和var使用类似，但是具有块级作用域。
const声明的变量不能再改变值，通常也称作常量，const一旦声明，就必须立即初始化，不能留到以后赋值。

let name = 'zs'



while (true) {

    let name = 'ls'

    console.log(name)  //ls

    break

}



console.log(name)  //zs



const a ;//报错,一旦声明变量，应该立即赋值

const b = 2;

b = 3 //报错，因为定义常量之后不能成重新赋值

class, extends, super

class关键字用来定义类,类似于ES5构造函数

extends关键字实现继承

super关键字，指代父类的实例（即父类的this对象）

子类必须在constructor方法中调用super方法，否则新建实例时会报错。这是因为子类没有自己的this对象，而是继承父类的this对象，然后对其进行加工。如果不调用super方法，子类就得不到this对象。

ES6的继承机制，实质是先创造父类的实例对象this（所以必须先调用super方法），然后再用子类的构造函数修改this

// 定义一个类

class Animal {

    // 构造函数

    constructor(){

        // 实例属性

        this.type = 'animal'

    }

    // 实例方法

    says(say){

        console.log(this.type + ' says ' + say)

    }

}



let animal = new Animal()

animal.says('hello') //animal says hello



// 定义一个类继承于另一个父类

class Cat extends Animal {

    constructor(){

        super()

        this.type = 'cat'

    }

}



let cat = new Cat()

cat.says('hello') //cat says hello

箭头函数

箭头函数没有自己的作用域

// ES5写法

function(x, y) { 

    x++;

    y--;

    return x + y;

}

// 箭头函数改写

(x, y) => {x++; y--; return x+y}



// ES5写法

// 没有参数

[1,2,3].map(function(){

    return 1;

})

// 改写

[1,2,3].map(()=>{return 1})或者[1,2,3].map(()=>1)



// 一个参数

[1,2,3].map(function(v){

    return v + 1;

})

// 箭头函数改写

[1,2,3].map(v =>{return v + 1});



// 一个参数

[1,2,3].map(function(v,i){

    return v + i;

})

// 箭头函数改写

[1,2,3].map((v,i) =>{return v + i});

// 箭头函数没有自己的作用预

var obj = {

    name:'zs',

    say:function(){

        setTimeout(function(){

            console.log(this.name);// 报错,此处this是window

        },0)

    }

}

obj.say();



// 箭头函数改写

var obj = {

    name:'zs',

    say:function(){

        setTimeout(()=>{

            console.log(this.name);// zs 箭头函数没有自己的作用预,延用上级作用预,this依旧是调用say方法的对象

        },0)

    }

}

obj.say();

其他特殊语法

对象中的属性名称和属性值如果同名，可以简写
对象中的方法可以省略function

var name = 'zs';

var obj = {

    name:name,

    say:function(){

        console.log('我的名字是' + this.name)

    }

}



// ES6

var name = 'zs';

var obj = {

    name, // 属性名和属性值同名

    say(){ // 省略:function

        console.log(`我的名字是${this.name}`)

    }

}
